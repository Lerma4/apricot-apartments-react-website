# Apricot Apartments

A high-fidelity *Cinematic Landing Page* for **Apricot Apartments** (Morsasco), built following the "Organic Tech" aesthetic preset.

This project is engineered not just as a website, but as a premium digital tool: every scroll is intentional, every animation (orchestrated via GSAP for peak performance) carries weight in the experience, rejecting generic designs in favor of "magnetic" micro-interactions.

## Tech Stack

- **Frontend Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v3 (Custom Configuration)
- **Animations**: GSAP 3 + ScrollTrigger
- **Iconography**: Lucide React
- **Routing**: React Router DOM (v6/v7)

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

For more structural details, interaction patterns, and file architecture, please refer to the [TECH_STACK.md](./TECH_STACK.md) file.
